# 🚀 FocusForge

> Smart Study Analytics & Productivity Tracker for Students

FocusForge is an intelligent Python-based productivity tracker designed to help students understand _how_ they study, not just _how long_ they study. It tracks study sessions, analyzes focus levels, generates productivity insights, and visualizes learning patterns using graphs and statistics.

---

# ✨ Features

✅ Add and manage study sessions  
✅ Track focus ratings and completion status  
✅ Analyze productivity trends  
✅ Generate smart study insights  
✅ Visualize data using graphs  
✅ JSON-based persistent data storage  
✅ Modular multi-file architecture  
✅ Input validation and error handling
